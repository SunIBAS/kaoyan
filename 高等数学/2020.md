> 二元函数极值

>> 1.求x偏导y偏导为0时的点

>> 2.求A=xx偏导，B=xy偏导，C=yy偏导，代入1中的x,y，

>> 若B*B-A*C<0，且A<0，为极大值，反之为极小值

>> 若B*B-A*C=0，可能为极值

![](./2020pic/二重积分求体积.jpg)

![](./2020pic/非方阵求解.jpg)

A = CTBC 矩阵合同，有相同的正负惯定数，如果A对称则B也对称

A = C-1BC 相似合同
```
（1）两者的秩相等；
（2）两者的行列式值相等；
（3）两者的迹数相等；
（4）两者拥有同样的特征值，尽管相应的特征向量一般不同；
（5）两者拥有同样的特征多项式；
（6）两者拥有同样的初等因子。
（7）若A与对角矩阵相似，则称A为可对角化矩阵，若n阶方阵A有n个线性无关的特征向量，则称A为单纯矩阵。
（8）相似矩阵具有相同的可逆性，当它们可逆时，则它们的逆矩阵也相似。
```
